== Description ==

WP Ultimate Gallery is an AWESOME minimalist gallery plugin for your WordPress website. This plugin creates unlimited gallery & not only gallery, you can enable those.

== Installation ==

Using this plugin is VERY easy. Lets do in three steps:
<ol>
 	<li>Just install via Plugin &gt; Add new &amp; search from WP directory. Or use upload method.</li>
 	<li>Create gallery, upload images &amp; save it! Get shortcode!</li>
 	<li>Use the shortcode on page, posts or widgets! Even via <code>do_shortcode</code> method ;)</li>
</ol>
That's all for today! Easy enough!


Contact with us if you are in any trouble. We are happy to help you.
